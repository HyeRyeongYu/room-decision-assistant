# Room Decision Assistant
2030세대 자취방 선택을 위한 의사결정 지원 웹앱 프로젝트

## Overview
본 프로젝트는 대학생과 사회 초년생이 주거 계약 과정에서 겪을 수 있는 정보 부족의 문제와 의사결정의 어려움을 해결하기 위해,  
체크리스트 기반 평가 및 비교 기능을 제공하는 웹앱을 개발하는 것을 목표로 한다.  
이를 통해 사용자가 객관적인 기준과 데이터를 바탕으로 최적의 자취방을 선택할 수 있도록 돕는다.

## Features
- 체크리스트 기반 항목 평가 기능
- 점수 계산을 통한 의사결정 지원 기능
- 여러 매물  비교 기능
- 사용자 맞춤형 체크리스트 항목 확장 기능
